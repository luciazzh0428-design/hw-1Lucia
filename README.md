# Homework 1: Coding Assistant Onboarding

This repository demonstrates my onboarding process using a coding assistant and GitHub workflow.

## Project Description
In this assignment, I used a coding assistant to generate an initial draft and then manually edited the content to create a final version. This process helped me understand how AI tools can support, but not replace, human input.
